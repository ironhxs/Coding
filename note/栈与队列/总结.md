# 栈与队列｜总结

关键：栈后进先出（LIFO）、队列先进先出（FIFO）、互相模拟、括号匹配、表达式求值。

## 常见模式

- 栈模拟队列
  - 双栈：输入栈 + 输出栈，输出栈空时才倒入
  - 典型题：232

- 队列模拟栈
  - 双队列：每次 pop/top 需将前 n-1 个元素移到另一个队列
  - 典型题：225

- 括号匹配
  - 遇到左括号入栈，遇到右括号检查栈顶是否配对
  - 典型题：20

- 相邻消除
  - 用栈或字符串模拟栈，相邻相同则弹出
  - 典型题：1047

- 表达式求值（逆波兰）
  - 栈存操作数，遇到运算符弹出两个操作数计算后压栈
  - 典型题：150

## 复杂度
- 时间：O(n)
- 空间：O(n)

## 易错点
- 栈为空时不能取 top
- 双栈/双队列倒入时机
- 运算符左右操作数顺序（先弹出的是右操作数）
